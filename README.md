# OMIEC-MD
This repository contains restart files for the MD models used in the paper "Dynamic Disorder as an Advantage for Charge Transport in in Organic Mixed Ionic-Electronic Conductors", submitted to [Journal] by K. Patrikar, C. Burke, and A. Troisi.

We provide LAMMPS torsional potential (.pot), input (.m), and data files (.data) for the polymer p(g2T-T). We provide GROMACS structure (.gro) and run files (.tpr) for polymers p(g2T-TT), P3MEET, and p(gDPP-T2).
